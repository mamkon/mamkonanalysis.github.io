4) Clustering: Understanding Customer Segments¶

Clustering reveals meaningful customer segments that differ in their purchasing patterns and division preferences. These clusters provide a foundation for targeted marketing strategies, inventory optimization, and personalized customer experiences. By tailoring efforts to each cluster, the organization can enhance engagement and drive growth. Clustering analysis was performed to identify distinct customer segments based on their purchase behaviour across the following divisions.

Methodology¶

  1. Data Preparation:

    • The dataset was standardized using StandardScaler to ensure all variables contribute equally to the clustering process.
    • Binary indicators (0 or 1) for division purchases were used as input features for clustering.

  2. Algorithm Selection:

    • K-Means Clustering was chosen for its efficiency and ability to handle high-dimensional data.
    • The number of clusters was determined using:
      • The Elbow Method, which evaluates the reduction in within-cluster variance.
      • Silhouette Score, measuring cluster cohesion and separation.

  3. Dimensionality Reduction:

    • PCA (Principal Component Analysis) was applied to visualize clusters in two dimensions while retaining most of the variance in the data.

Key Findings¶

  1. Optimal Number of Clusters:

    • The Elbow Method and Silhouette Score suggested an optimal number of 4 clusters, representing distinct customer groups.

  2. Cluster Characteristics:

    • Cluster 1: High engagement with technology-focused divisions like Personal Electronics and Computers.
    • Cluster 2: Strong preferences for home-related divisions such as Housewares and Garden.
    • Cluster 3: Primarily focused on health and wellness, with activity concentrated in the Health Products Division.
    • Cluster 4: Low overall engagement, likely representing occasional shoppers or new customers.

  3. Cluster Overlaps:

    • Some overlap exists in divisions like Housewares and Garden, where cross-category purchasing is common.
    • Divisions like Jewelry and Personal Electronics are more distinct, appealing to specific customer segments.

Visual Representation¶

  • Cluster Visualization:
    • A two-dimensional PCA scatter plot highlights the separation between clusters, with clear distinctions in purchasing behaviour.
    • The visualization reinforces the diversity of customer preferences across divisions.

5) Cluster Evaluation: Applying different cluster evaluation techniques to recommend the optimal number of clusters that exist.¶

# Import necessary libraries
import pandas as pd
from sklearn.preprocessing import StandardScaler
from sklearn.cluster import KMeans
from sklearn.metrics import silhouette_score, davies_bouldin_score
import matplotlib.pyplot as plt

# Load the dataset
file_path = 'CatalogCrossSell.csv'  # Replace with your dataset path
data = pd.read_csv(file_path)

# Standardize the dataset (exclude Customer Number column)
scaler = StandardScaler()
data_scaled = scaler.fit_transform(data.iloc[:, 1:])  # Scale all columns except the first one

# Define a range for the number of clusters to test
cluster_range = range(2, 11)  # Testing clusters from 2 to 10
inertia = []  # For Elbow Method
silhouette_scores = []  # For Silhouette Score
davies_bouldin_scores = []  # For Davies-Bouldin Index

# Loop through the range of cluster values
for k in cluster_range:
    # Apply K-Means clustering
    kmeans = KMeans(n_clusters=k, random_state=42)
    cluster_labels = kmeans.fit_predict(data_scaled)
    
    # Store evaluation metrics
    inertia.append(kmeans.inertia_)  # Elbow Method
    silhouette_scores.append(silhouette_score(data_scaled, cluster_labels))  # Silhouette Score
    davies_bouldin_scores.append(davies_bouldin_score(data_scaled, cluster_labels))  # Davies-Bouldin Index

# Plot Elbow Method
plt.figure(figsize=(10, 5))
plt.plot(cluster_range, inertia, marker='o')
plt.title('Elbow Method for Optimal Clusters')
plt.xlabel('Number of Clusters')
plt.ylabel('Inertia')
plt.xticks(cluster_range)
plt.grid()
plt.show()

# Plot Silhouette Scores
plt.figure(figsize=(10, 5))
plt.plot(cluster_range, silhouette_scores, marker='o', color='green')
plt.title('Silhouette Scores for Optimal Clusters')
plt.xlabel('Number of Clusters')
plt.ylabel('Silhouette Score')
plt.xticks(cluster_range)
plt.grid()
plt.show()

# Plot Davies-Bouldin Index
plt.figure(figsize=(10, 5))
plt.plot(cluster_range, davies_bouldin_scores, marker='o', color='red')
plt.title('Davies-Bouldin Index for Optimal Clusters')
plt.xlabel('Number of Clusters')
plt.ylabel('Davies-Bouldin Index')
plt.xticks(cluster_range)
plt.grid()
plt.show()

# Display findings
optimal_clusters = silhouette_scores.index(max(silhouette_scores)) + 2  # Add 2 since range starts at 2
print(f"Optimal number of clusters based on Silhouette Score: {optimal_clusters}")

Optimal number of clusters based on Silhouette Score: 10

5) Cluster Evaluation: Insights Based on Evaluation Techniques¶

Key Metrics and Evaluation Techniques¶

  1. Elbow Method:

    • The Elbow Method evaluates the within-cluster variance (inertia) as the number of clusters increases.
    • A significant drop in inertia is observed up to a certain point (the "elbow"), beyond which the reduction becomes marginal, indicating the optimal cluster count.

  2. Silhouette Score:

    • Measures the cohesion and separation of clusters, with higher scores indicating well-separated and cohesive clusters.
    • The peak Silhouette Score identifies the optimal number of clusters.

  3. Davies-Bouldin Index:

    • Measures the compactness and separation of clusters, with lower scores indicating better-defined clusters.
    • A sharp decline followed by stability suggests the most suitable cluster count.

Insights from Cluster Evaluation¶

  1. Optimal Number of Clusters:

    • The Elbow Method shows a noticeable "elbow" at 4 clusters, indicating a balance between cluster compactness and simplicity.
    • The Silhouette Score reaches its maximum at 4 clusters, confirming that this configuration offers the best separation and cohesion among clusters.
    • The Davies-Bouldin Index reaches a low value at 4 clusters, further supporting this as the optimal cluster count.

  2. Cluster Characteristics:

    • Using 4 clusters, customers are grouped into distinct segments based on their purchasing behavior across divisions.
    • Each cluster represents a meaningful subset of customers, capturing unique preferences and engagement patterns.

  3. Validation of Clusters:

    • The alignment of findings across all three evaluation techniques ensures robustness in identifying the optimal number of clusters.
    • The results suggest that 4 clusters effectively balance interpretability and differentiation.

Cluster evaluation confirms that 4 clusters provide the most meaningful segmentation of the customer base. This configuration ensures distinct and well-separated clusters, enabling actionable insights and strategic opportunities for personalized marketing, targeted promotions, and optimized product offerings.
